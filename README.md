# Boxd Card

A Chrome extension that generates a shareable **Last Four Watched** image card from your Letterboxd profile.

Open the extension on any `letterboxd.com/<username>` profile page, click **Generate Card**, and get a 1200×560 PNG you can download or copy straight to your clipboard.

Custom posters (a Letterboxd Pro/Patron feature) are supported automatically — they're already rendered in the DOM by the time the extension reads them, so no API access is required.

## How it works

The extension reads film metadata and poster images directly from the Letterboxd profile page DOM — no backend, no API calls, no CORS workarounds. Poster images are fetched cross-origin by the background service worker (which has the necessary host permissions) and drawn to an HTML5 Canvas to produce the final PNG.

```
Letterboxd profile page DOM
        │  chrome.runtime.sendMessage
        ▼
  Content Script       reads titles, ratings, poster URLs, username
        │  message response
        ▼
  Popup UI             drives the generation flow
        │
        ├─► Background Service Worker   fetches poster image blobs
        │
        ▼
  Canvas Renderer      draws the 1200×560 card
        │
        ▼
  Download PNG  /  Copy to clipboard
```

Each element is individually toggleable from the popup: film title, year, star rating, and card date.

## Development

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev      # build in watch mode → dist/
npm run build    # one-shot production build
npm run test     # Vitest in watch mode
npm run test:run # single test run (CI)
npm run coverage # v8 coverage report
```

**Loading the extension in Chrome:**

1. Run `npm run build`
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** → select the `dist/` folder

After each subsequent `npm run build`, click the **↺ reload** button on the extension card. For content script changes, also refresh the Letterboxd tab.

## Tech stack

- **Chrome Manifest V3** — service worker, content script, popup
- **React + TypeScript** — popup UI
- **Vite + @crxjs/vite-plugin** — build tooling
- **Canvas 2D API** — card image generation (no external image library)

## Permissions

| Permission | Reason |
|---|---|
| `activeTab` | Read the URL of the current tab to validate it's a Letterboxd profile page |
| `clipboardWrite` | Copy the generated card image to the clipboard |
| `https://letterboxd.com/*` | Fetch poster redirects from the Letterboxd CDN |
| `https://a.ltrbxd.com/*` | Fetch poster images from the primary Letterboxd image CDN |
| `https://s.ltrbxd.com/*` | Fetch static assets (logo) |
