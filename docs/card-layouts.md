# Card Layout Reference

Boxd Card supports six layout formats. Each layout defines the canvas dimensions, poster grid arrangement, and text positioning for generated card images. The layout preference is set in the extension settings and applies to all card types.

All layouts share a common structure:

- **Header** (90px): Letterboxd logo (left) and date (right)
- **Title area** (variable): card type label, list title/description, or tags
- **Poster grid**: film posters with title, year, rating, and optional date
- **Footer** (64px): username/avatar (left) and "generated by Boxd Card" (right)

Canvas width is always fixed per layout. Height is either fixed (for small film counts) or dynamic (for larger grids).

Dynamic height. Poster beside review text (side-by-side).

- Poster: 200 x 300 px at x=40
- Text column starts at x=270 (poster + 30px gap), width=890px
- Font sizes: title 39px, meta 32px, review 30px (line height 42px)
- Multiple reviews stack vertically with 28px gap

## Square

| Property | Value |
|----------|-------|
| Canvas size | 1080 x 1080 px (fixed for 1-4 films) |
| Best for | Instagram grid posts |

### 1-4 films

2x2 grid (or 1x1/1x2 for fewer films). Text below each poster. Poster height scales to fill available vertical space.

- Poster: ~398 x ~265 px (derived from available space)
- Gap: 20 px
- Centered horizontally

### 5-20 films

3-column grid, dynamic height.

- Poster: 320 x 480 px
- Gap: 20 px
- Text below each poster

### Review card

Fixed 1080px height. Side-by-side layout (poster left, text right).

- Poster: 160 x 240 px
- Font sizes: title 32px, meta 26px, review 26px (line height 36px)
- Long reviews are clipped at the bottom edge

## 4:5

| Property | Value |
|----------|-------|
| Canvas size | 1080 x 1350 px (fixed for 1-4 films) |
| Best for | Instagram feed posts |

### 1-4 films

Same grid arrangement as Square (2x2), but with 270px more vertical space. Posters scale taller to fill the extra height.

- Poster: ~453 x ~302 px (derived from available space)
- Gap: 20 px
- Text below each poster

### 5-20 films

Identical to Square: 3-column grid, 320x480 posters, dynamic height.

### Review card

Fixed 1350px height. Same styling as Square (side-by-side, 160x240 poster). More vertical space allows longer reviews before clipping.

## Common Behaviors

### Text rendering (poster-grid cards)

Below each poster, up to four text items are drawn in order:

1. **Title** (bold, white, 24px) — word-wrapped to max 2 lines, truncated with ellipsis if needed
2. **Year** (gray, 24px) — shown when "Show Year" is enabled (requires title)
3. **Rating** (amber/gold, 24px) — star characters (★★★½)
4. **Watch date** (gray, 21px) — only for Recent Diary card type

### Title area

Optional content drawn between the header and poster grid:

- **Card type label** (bold, 36px) — e.g. "Last Four Watched", "Favorites"
- **List title** (bold, 32px) — list card type only
- **List description** (24px) — list card type only
- **Tag pills** — list and review card types

The title area shifts the poster grid and footer down uniformly.

### Footer

Two modes:

- **Single user**: `[avatar] username` (left), `generated by Boxd Card` (right)
- **Attribution**: `[your-avatar] you 🔗 [author-avatar] author` (left) — shown when generating a card from someone else's profile

### Backdrop

Review and List card types support an optional backdrop image. When provided:

1. The backdrop is drawn blurred (20px blur, oversized by 3x blur radius to hide edge artifacts)
2. A dark overlay (`rgba(0,0,0,0.72)`) is applied
3. All other content draws on top

Falls back silently to the solid background color (`#1a1a1a`) if the image fails to load.

### Poster sizing

All poster dimensions maintain a 2:3 aspect ratio (width:height). For fixed-height layouts, poster height is derived from available vertical space to maximize the use of the canvas.

### Film count thresholds

| Layout | Fixed (≤4 films) | Grid (5-20 films) |
|--------|-------------------|---------------------|
| Landscape | 1-row, 4-col | 5-col, variable height |
| Square | 2x2 | 3-col, variable height |
| 4:5 | 2x2 | 3-col, variable height |
| 3:4 | 2x2 | 3-col, variable height |
| Story | 2x2 (1-film showcase) | 2-col, variable height |
| Banner | 1-row, up to 4-col | 5-col, variable height |
