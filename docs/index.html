No two alike

Every card gets its own gradient.

Each render samples two of Letterboxd's six brand colours at a random angle, then sits them under a near-black wash. The result is a subtle, one-of-a-kind backdrop — loud enough to feel alive, quiet enough to keep the posters in front.

How it works

Same card generator, two ways to run it — about ten seconds either way.

Chrome extension

Generate from the page you're on

01

Open the popup

On any Letterboxd profile, diary, list, or review page — it reads the page you're already viewing, no URL needed.

02

Pick your options

Choose the card type, layout, and display details — film count, titles, years, ratings.

03

Download & share

Save the PNG, copy it to your clipboard, or share straight to stories and posts.

Web app

No install — paste a link

01

Paste a URL

Drop in any Letterboxd profile, diary, list, or reviews link at boxd-card.com/app — it detects the type automatically.

02

Pick your options

The same card types, layouts, and display details as the extension.

03

Download & share

Save or copy the PNG right from the browser — nothing to install.

Both produce the same PNG, but a few things are extension-only: Stats cards (Letterboxd blocks stats pages from outside services, so the web app can't fetch them), Remember my username (the extension caches it locally for quicker navigation), and custom posters with TMDB (the extension keeps a film's custom Letterboxd poster when enriching from TMDB; the web app uses the TMDB poster if enrichment is enabled). For those, use the Chrome extension. Everything else works in both.

Manual install

Prefer to sideload?

Grab the latest ZIP from the Releases page, unpack it, then load it in Chrome:

  1. Open chrome://extensions
  2. Enable Developer mode (top-right toggle)
  3. Click Load unpacked and select the folder

Sideloaded installs don't auto-update — the popup shows an update available banner whenever a newer release ships.
