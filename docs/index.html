Boxd Card

A Chrome extension that generates a shareable image card from your Letterboxd profile.

Don't have a Chromium browser to run the extension? Try the Boxd Card web app — generate cards directly in your browser on any device, including mobile.

How it works

Navigate to your Letterboxd profile, diary, or a list — then open the Boxd Card extension and click Generate. Your card is composed locally in the browser and can be copied to your clipboard or downloaded as a PNG. No account, no login, no data collection.

Card types

  • Last Four Watched — your four most recent films from your profile page
  • Favorites — your pinned favorite films
  • Recent Diary — up to 20 entries from your diary
  • List — up to 20 films from any public list
  • Review — 1–4 reviews from your reviews page or a single film review

Installation

Install from the Chrome Web Store — one click, auto-updates, works in Chrome and any Chromium-based browser that supports Web Store extensions.

Prefer to sideload manually? Grab the latest ZIP from the Releases page, unpack it, then load the extension in Chrome:

  1. Open chrome://extensions in Chrome.
  2. Enable Developer mode (top-right toggle).
  3. Click Load unpacked and select the unpacked extension folder.

Sideloaded installs don't auto-update, but the extension popup will show an update available banner whenever a newer release is published so you know when to grab a fresh ZIP.
